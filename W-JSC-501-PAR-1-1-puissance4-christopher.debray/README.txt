## 2 Fonctionnement
La class game servira de réceptacle à toutes les fonction.
Le constructor de celle-ci sera utilisée pour récupérer les valeurs utilisées par la personne, exemple : couleurs des pions, taille de la grille, etc
Ces valeurs auront chacunes une valeurs par défaut.
